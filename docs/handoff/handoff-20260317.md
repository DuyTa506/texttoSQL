## Key Decisions Made

| Decision | Rationale |
|---|---|
| `GraphRetriever` moved to `retrieval/` not left in `schema_graph/` | Role is Phase 2 schema-linking, not graph construction |
| `BaseRetriever` ABC includes `value_matches` param | Allows uniform call site in `run_pipeline.py` regardless of backend |
| `HybridRetriever` accepts but ignores `value_matches` | Interface compat today; easy to add value-hint support later |
| `embed_nodes()` accepts `BaseEmbeddingModel \| str` | Both HF and OpenAI embedding providers work without changing callers |
| `batch_size` removed from `embed_nodes()` signature | Belongs to the model instance, not the calling code |
| Phase 3 structural refactors deferred | Retrieval API not yet stable (v3 algorithm still in progress) |
